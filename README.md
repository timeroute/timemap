# TimeMap

👏🏻 Build a map based on WebGL. 

Thanks to the article [Building a Vector Map from Scratch](https://ckochis.com/building-a-vector-map-from-scratch).

## Features

🏠 Has Features
- Matrix transform. traslate, scale, rotate.
- Load vector tile.
- Load geojson.
- Load image tile.
- Fly to.
- Support mobile gesture.
